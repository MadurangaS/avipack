# Generic Brain Only

This starter adds Avipack Brain to a new or existing project without choosing an application framework.

It is the primary foundation-stage template because many users will already have a project and only need structured project memory, change tracking, ADRs, and agent rules.

The active MVP template used by `avipack init` now lives in `packages/core/templates/generic-brain-only/template` so it can be bundled with `@avipack/core` for future npm publishing.
