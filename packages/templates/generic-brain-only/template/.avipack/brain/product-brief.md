# Product Brief

## Summary

This project uses Avipack Brain to keep product intent, requirements, architecture, decisions, and change requests visible to humans and coding agents.

## Users

- Project owner
- Developers
- Reviewers
- Approved coding agents

## Current Scope

- Establish the project brain.
- Record initial requirements.
- Track architecture decisions.
- Require explicit change requests for meaningful scope changes.

## Out of Scope

- Autonomous background agents.
- Secret storage inside brain files.
- Unreviewed architecture changes.
