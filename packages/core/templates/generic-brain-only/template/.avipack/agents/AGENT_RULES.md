# Agent Rules

These rules apply to coding agents working inside this generated Avipack project.

1. Read `.avipack/brain/project.yaml` before making project-wide changes.
2. Read `.avipack/brain/requirements.yaml` before changing behavior.
3. Update requirements, ADRs, or change requests when behavior or architecture changes.
4. Do not store secrets in `.avipack`.
5. Do not run bots or tools that modify files unless the owner explicitly requested that action.
6. Keep changes small and reviewable.
7. Respect bot and agent write boundaries.
8. Prefer clear reports over silent rewrites.
