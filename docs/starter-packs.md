# Starter Packs

Starter packs create project structures. Avipack supports both full application starters and brain-only adoption.

## `generic-brain-only`

Status: scaffolded.

Adds `.avipack`, `avipack.config.yaml`, starter requirements, ADRs, change requests, schemas, reports, and agent rules. This is the MVP priority because users may already have a project.

## `nextjs-typescript-postgres`

Status: planned.

Future starter for a Next.js app with TypeScript and PostgreSQL.

## `node-express-typescript`

Status: planned.

Future starter for a Node.js API with Express and TypeScript.

## `python-fastapi`

Status: planned.

Future starter for a Python API with FastAPI.

## Template Variables

Future versions should support project name, package manager, language, framework, database, license, and owner metadata.
